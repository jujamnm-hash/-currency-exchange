## 2️⃣ پاککردنەوەی Cache (گرنگترین هەنگاو!)

### لە Chrome/Edge:
1. کرتە لەسەر **F12** (Developer Tools)
2. کرتە لەسەر **Network** tab
3. کرتە لەسەر **Disable cache** ✓
4. کرتە لەسەر **F5** یان **Ctrl+Shift+R**

یان:

1. **Ctrl + Shift + Delete**
2. هەڵبژێرە: **Cached images and files**
3. کرتە لەسەر **Clear data**

### لە مۆبایل (Chrome):
1. بڕۆ بۆ: **⋮** → **Settings**
2. **Privacy** → **Clear browsing data**
3. هەڵبژێرە: **Cached images and files**
4. کرتە لەسەر **Clear data**

## 5️⃣ تاقیکردنەوەی تایبەتمەندییەکان

### تاقیکردنەوەی نرخی راستەوخۆ (Live Rates):
1. کرتە لەسەر **⚡**
2. کرتە لەسەر **📡 نرخی راستەوخۆ**
3. چالاک بکەرەوە
4. **نوێکردنەوە ئێستا** کرتە بکە
5. دەبێت نرخەکانی USD, EUR, GBP, TRY, AED ببینیت

### تاقیکردنەوەی چارتی پێشکەوتوو:
1. کرتە لەسەر **⚡**
2. کرتە لەسەر **📊 چارتی پێشکەوتوو**
3. هەڵبژێرە: Area Chart یان Heatmap

### تاقیکردنەوەی فیلتەر:
1. کرتە لەسەر **⚡**
2. کرتە لەسەر **🔍 فیلتەر**
3. فیلتەرەکان پڕبکەرەوە
4. **جێبەجێکردن** کرتە بکە

### تاقیکردنەوەی Excel Export:
1. کرتە لەسەر **⚡**
2. کرتە لەسەر **📊 Excel**
3. فایلێک داگیرێت (CSV format)

### تاقیکردنەوەی بەراوردکردن:
1. کرتە لەسەر **⚡**
2. کرتە لەسەر **⚖️ بەراوردکردن**
3. هەڵبژێرە: بەراوردی مانگەکان یان دراوەکان

## 6️⃣ چارەسەرکردنی کێشەکان

### ئەگەر دوگمەی ⚡ نییە:
1. **F12** لە کیبۆرد بکە
2. بڕۆ بۆ **Console** tab
3. بگەڕێ بۆ هەڵەکان (رەنگی سوور)
4. ئەگەر دەبینیت:
   - `Failed to load ultimate-features.js` → Cache پاک بکەرەوە
   - `Failed to load advanced-pro-features.js` → Cache پاک بکەرەوە

### ئەگەر تایبەتمەندییەک کار ناکات:
1. **F12** بکە
2. لە **Console** وردبەرەوە
3. ئەگەر دەبینیت:
   - `showAnalytics is not defined` → فایلەکان لۆد نەبوون
   - `TypeError` → هەڵەیەکی کۆد

**چارەسەر:**
1. Cache پاک بکەرەوە (Hard Reload)
2. پەڕەکە دابخەرەوە
3. چاوەڕوانی 5 چرکە بکە
4. دووبارە تاقی بکەرەوە

### ئەگەر نرخی راستەوخۆ کار ناکات:
- **هۆکار:** API سەرقاڵە یان بلۆک کراوە
- **چارەسەر:** دواتر تاقی بکەرەوە
