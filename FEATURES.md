- شیکاری بەپێی دراو

### 3️⃣ بەڕێوەبردنی پۆرتفۆڵیۆ (Portfolio Management) 💼

- بەڕێوەبردنی سەربەخۆ بۆ هەر پۆرتفۆڵیۆیەک

**بەکارهێنان:**
- پۆرتفۆڵیۆی کار
- پۆرتفۆڵیۆی کەسی
- پۆرتفۆڵیۆی خێزانی

## 📱 چۆنیەتی بەکارهێنان

### کردارە خێراکان ⚡

1. **کلیک لە دوگمەی ⚡** لە سەرەوە
2. **هەڵبژاردنی تایبەتمەندی** کە دەتەوێت
3. **بەکارهێنان و چێژوەرگرتن!**

### پۆرتفۆڵیۆکان 💼

1. کلیک لە ⚡ → پۆرتفۆڵیۆکان
2. زیادکردنی پۆرتفۆڵیۆی نوێ
3. گۆڕین لە نێوانیان بە ئاسانی

### قاڵبەکان 📋

1. فۆرمی گۆڕینەوە پڕ بکەرەوە
2. کلیک لە ⚡ → قاڵبەکان
3. پاشەکەوتی وەک قاڵب
4. بەکارهێنان لە جارێکی تر بە خێرایی

### ئامانجەکان 🎯

1. کلیک لە ⚡ → ئامانجەکان
2. ➕ ئامانجی نوێ
3. ناو، بڕ، بەروار بنووسە
4. شوێنکەوتنی پێشکەوتنت

### گەڕان 🔍

1. کلیک لە ⚡ → گەڕان
2. فلتەرەکان هەڵبژێرە
3. کلیک لە "گەڕان"
4. ئەنجامەکان دەبینیت
