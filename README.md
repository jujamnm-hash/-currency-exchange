# 💱 سیستەمی پێشکەوتووی گۆڕینەوەی دراو

سیستەمێکی تەواو و پێشکەوتوو بۆ تۆماری گۆڕینەوەکانی دراو و حسابکردنی قازانج و زەرەر لەگەڵ تایبەتمەندییە زیرەکەکان.

## 🌟 تایبەتمەندییە نوێیەکان

### ✨ تایبەتمەندییە سەرەکییەکان

#### 1. **پشتگیری چەندین دراو**
- 💵 دۆلاری ئەمریکی (USD)
- 💶 یۆرۆ (EUR)
- 💷 پاوەندی بەریتانی (GBP)
- 💴 لیرەی تورکی (TRY)
- 💵 دیرهەمی ئیماراتی (AED)

#### 2. **هێڵکاری و شیکاری زیرەک**
- 📈 هێڵکاری قازانج و زەرەر بەدرێژایی کات
- 📊 هێڵکاری باڵانسی دراوەکان
- 📉 هێڵکاری نرخەکان بەدرێژایی کات
- شیکاری ئۆتۆماتیک بە Chart.js

#### 3. **ڕەنگی تاریک/ڕووناک**
- 🌙 دۆخی تاریک بۆ بەکارهێنان لە شەو
- ☀️ دۆخی ڕووناک بۆ ڕۆژ
- گۆڕینی ئاسان بە یەک کرتە

#### 4. **ئامانجی نرخ**
- 🎯 دانانی ئامانجی نرخ بۆ هەر گۆڕینەوەیەک
- 🔔 ئاگادارییەکان کاتێک نرخ دەگاتە ئامانج
- بینینی ئامانجەکان لە بەشی تایبەت

#### 5. **هەناردە و هاوردەی زانیاری**
- 📥 هەناردەی زانیاری وەک JSON
- 📊 هەناردەی زانیاری وەک CSV بۆ Excel
- 📄 هەناردەی وەک PDF (لە ڕێگەی چاپکردن)
- 📤 هاوردەی زانیاری لە فایلی JSON

#### 6. **ئاماری پێشکەوتوو**
- کۆی گۆڕینەوەکان
- کۆی دراوی کڕاو و فرۆشراو
- دراوی ماوە
- تێچووی نێوەند
- **[نوێ]** باشترین و خراپترین گۆڕینەوە
- **[نوێ]** ڕێژەی سەرکەوتن

#### 7. **ڕێکخستنەکان**
- دیاریکردنی دراوی بنەڕەت
- چالاک/ناچاڵککردنی ئاگادارییەکان
- پاشەکەوتی ئۆتۆماتیک

## چۆنیەتی بەکارهێنان

### 1. زیادکردنی گۆڕینەوە

1. جۆری گۆڕینەوە هەڵبژێرە (کڕین یان فرۆشتن)
2. بڕی دۆلار بنووسە
3. نرخی گۆڕینەوە بنووسە
4. بەرواری گۆڕینەوە هەڵبژێرە
5. تێبینی زیاد بکە (ئەگەر دەتەوێت)
6. دوگمەی "زیادکردن" دابگرە

### 2. بینینی مێژوو

- هەموو گۆڕینەوەکانت لە خوارەوە دەبینیت
- دەتوانیت هەر گۆڕینەوەیەک بسڕیتەوە

### 3. حسابکردنی نرخی ئێستا

1. نرخی ئێستای دۆلار بنووسە
2. دوگمەی "حساب بکە" دابگرە
3. ئەنجامەکەت پیشان دەدرێت:
   - چەند قازانجت یان زەرەرت دەبێت
   - چەند لەسەدا قازانج/زەرەرە
   - ئایا باشترە بفرۆشیت یان چاوەڕێ بکەیت

## چۆنیەتی داگرتن لەسەر GitHub

### پێداویستییەکان
- هەژمارێکی GitHub (بەخۆڕایی دروست بکە لە [github.com](https://github.com))

### هەنگاوەکان

#### 1. دروستکردنی Repository نوێ

1. بڕۆ بۆ [github.com](https://github.com)
2. کلیک لە دوگمەی **+** لە سەرەوە بکە
3. **New repository** هەڵبژێرە
4. ناوێک بۆ repository بنووسە (بۆ نموونە: `currency-exchange`)
5. **Public** هەڵبژێرە
6. کلیک لە **Create repository** بکە

#### 2. بارکردنی فایلەکان

1. لە پەڕەی repository، کلیک لە **uploading an existing file** بکە
2. هەموو فایلەکان هەڵبژێرە:
   - `index.html`
   - `style.css`
   - `app.js`
   - `manifest.json`
   - `README.md`
3. کلیک لە **Commit changes** بکە

#### 3. چالاککردنی GitHub Pages

1. لە repository، بڕۆ بۆ **Settings**
2. لە مێنووی لای چەپەوە، کلیک لە **Pages** بکە
3. لەژێر **Source**، هەڵبژێرە **main** branch
4. کلیک لە **Save** بکە
5. دوای چەند خولەکێک، لینکی ئەپەکەت پیشان دەدرێت

#### 4. بەکارهێنانی ئەپەکە لەسەر مۆبایل

لینکەکە دەبێتە شێوەی:
```
https://your-username.github.io/currency-exchange/
```

- ئەم لینکە لە وێبگەڕی مۆبایلەکەتدا بکەرەوە
- دەتوانیت وەک shortcut لە Home Screen زیادی بکەیت
- کاردەکات وەک ئەپێکی ڕاستەقینە!
